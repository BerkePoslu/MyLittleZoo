About

HybridZoo is a creative and imaginative project that depicts a fictional zoo located in Lausanne, Switzerland. The project is unique in that it features hybrid animals that are created by blending two different species. This open source school project was developed as a fun and educational way to explore genetics and the possibilities that arise when combining different animal traits.
